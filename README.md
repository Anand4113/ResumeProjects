## Featured Projects

### 1. AI MRI Denoiser
**Folder:** `ai-mri-denoiser/`

Deep learning project focused on denoising brain MRI images using U-Net and Autoencoder architectures. Includes model training scripts, inference scripts, sample outputs, and project documentation.

**Skills:** TensorFlow, CNNs, image preprocessing, computer vision, model evaluation

### 2. Diabetes Detection Prediction
**Folder:** `diabetes-detection-prediction/`

Classification project using health and lifestyle variables to predict diabetes risk. The project compares logistic regression and random forest models on a large structured dataset.

**Skills:** classification, statistical modeling, model comparison, healthcare analytics

### 3. School Discipline Impact Analysis
**Folder:** `school-discipline-impact-analysis/`

Data and society research project analyzing discipline patterns in schools and their broader social context, with emphasis on causal reasoning, interpretation, and communication.

**Skills:** research design, social impact analysis, reporting
